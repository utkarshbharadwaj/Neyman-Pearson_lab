    This tool covers nine classical one-parameter families, chosen because each has a sufficient statistic with a closed-form sampling distribution: Normal mean & variance, Exponential rate, Poisson mean, Binomial p, Gamma rate, Uniform upper bound, Lognormal log-mean, and Beta(α,1) shape. Critical values for discrete statistics (Poisson, Binomial) use a randomized boundary test to hit the target α exactly - shown as a split-coloured bar. Two-sided UMPU tests are solved numerically from the unbiasedness condition β′(θ₀) = 0 for continuous families; for Poisson/Binomial and the Uniform boundary parameter, an exact two-sided UMPU test requires further randomized-boundary machinery this tool flags rather than approximates silently.
