    This tool covers nine classical one-parameter families, chosen because each has a sufficient statistic with a closed-form sampling distribution. Critical values for discrete statistics (Poisson, Binomial) use a randomized boundary test to hit the target α exactly - shown as a split-coloured bar. Two-sided UMPU tests are solved numerically from the unbiasedness condition β′(θ₀) = 0 for continuous families; exact two-sided UMPU tests for discrete families are flagged.
